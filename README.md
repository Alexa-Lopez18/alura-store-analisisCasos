# Análisis de Tiendas - Alura Store

Proyecto de análisis de datos realizado con Python y Google Colab para evaluar el rendimiento de las tiendas de Alura Store.

## Análisis realizados

- Facturación total por tienda
- Categorías más vendidas
- Evaluación promedio de clientes
- Productos más y menos vendidos
- Costo promedio de envío

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Objetivo

Identificar qué tienda presenta menor rendimiento para recomendar su venta e invertir en un nuevo negocio.
